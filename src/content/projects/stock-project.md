# Building the Stock Project

This project was built to solve the complex problem of algorithmic stock prediction and visualization. 

I needed a robust dashboard to manage data pipelines, execute optimizer tests, and view output seamlessly. The entire UI is built with React and heavily utilizes advanced state management for rendering complex UI elements.

## The Challenge

The hardest part was connecting the Python backend Opportunity Generator with the React frontend so it looked beautiful and felt highly responsive.
